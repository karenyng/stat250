  • Consumer Price Index HTML form
    The form is at http://www.rateinflation.com/consumer-price-index/usa-historical-cpi In this simple example, we can select a different starting year and submit the form and see the URL in the navigation bar of our Web browser: 
    http://www.rateinflation.com/consumer-price-index/usa-historical-cpi?start-year=1989&end-year=2014
    This illustrates the two parameters for the form and their names - start-year and end-year. In other cases, there may be more that are not apparent from the HTML display but are in the HTML description of the form. In other cases, we may have to POST the request and the parameters will be sent in the body of the request, not in the URL.

    We can request the data with 

    u =  'http://www.rateinflation.com/consumer-price-index/usa-historical-cpi?start-year=1989&end-year=2014'
txt = getForm(u)
    Then we can parse this as an HTML document and then read the table(s) in it: 
    cpi = readHTMLTable( htmlParse(txt) )
    For more complex cases (e.g. POST requests, where there are many parameters, some hidden), we can use the RHTMLForms package to read the description of a form from the HTML document. Then we can programmatically generate an R function from this description and then invoke this function with inputs from R. See Lec14.Rsession.
